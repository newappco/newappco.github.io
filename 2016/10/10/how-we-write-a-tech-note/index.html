Why we use Markdown to write our notes

It’s easy to write a note by Markdown syntax, especially we are all technical guys.

Here are the common Markdown syntax

1) Italic text

*This is italic text *
Result
This is italic text

2) Bold text

**This is bold text**
Result
This is bold text

3) Strikethrough

~~This is fucking shit~~
Result
This is fucking shit

4) Headers - use #

This is # = h1
This is ## = h2
This is ### = h3
This is #### = h4
This is ##### = h5
This is ###### = h6

5) Images and Links

Image format: ![Alt text](url)

Link format: [Title text](url)
